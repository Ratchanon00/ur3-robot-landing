🧠 AI Integration

AI Features
ที่ขับเคลื่อนระบบ

สามโมดูล AI หลักที่ทำให้หุ่นยนต์ UR3 สามารถทำงานได้อย่างอัตโนมัติ และอัจฉริยะโดยไม่ต้องพึ่งการควบคุมจากมนุษย์

01
🎯

Object Detection

AI ตรวจจับกล่องสินค้าที่ยังไม่พับในพื้นที่ทำงาน คำนวณตำแหน่งที่แน่นอน และส่งคำสั่งให้แขนกล UR3 เข้าหยิบได้อย่างแม่นยำทุกจุดในระยะการทำงาน

  • ตรวจจับกล่องสินค้าที่ยังไม่พับ
  • คำนวณพิกัด 3D สำหรับแขนกล
  • ระบุตำแหน่งมุมและขอบกล่อง
  • รองรับกล่องหลายขนาดพร้อมกัน
02
🔄

Side Detection & Verification

AI ตรวจสอบว่ากล่องหงายถูกฝั่งหรือผิดฝั่ง ก่อนที่แขนกลจะเริ่มพับ เพื่อให้ผลลัพธ์ถูกต้อง และป้องกันความเสียหายในกระบวนการ

  • ตรวจสอบฝั่งถูก/ผิดของกล่อง
  • จำแนก orientation ของกล่อง
  • แจ้งเตือนเมื่อกล่องวางผิดฝั่ง
  • ปรับทิศทางการพับให้ถูกต้อง
03
🔤

Symbol & Text Recognition

AI อ่านและแปลความหมายสัญลักษณ์ ตัวหนังสือ บนกล่องสินค้า เพื่อช่วยให้แขนกล UR3 สามารถหยิบกล่องได้ถูกต้องทุกจุดในระยะทำงาน

  • อ่านสัญลักษณ์และตัวอักษรบนกล่อง
  • ระบุประเภทและขนาดกล่องจากสัญลักษณ์
  • OCR สำหรับข้อความบนบรรจุภัณฑ์
  • แยกแยะกล่องตามสัญลักษณ์ที่อ่านได้
🤖
AI Integration Architecture

ทั้ง 3 โมดูล AI ทำงานร่วมกันแบบ Real-time Pipeline — Object Detection ค้นหาและระบุตำแหน่งกล่อง → Side Verification ยืนยันฝั่งที่ถูกต้อง → Symbol Recognition อ่านข้อมูลบนกล่อง → UR3 Execution แขนกลพับกล่องอัตโนมัติ

● Real-time Processing
● End-to-End Automation
● Intelligent Decision Making
